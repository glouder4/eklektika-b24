# ACTION: `UPDATE_GROUP`

Обновление группы пользователей главного модуля (`CGroup::Update`).

## Обязательные поля тела запроса

| Поле | Тип | Описание |
|------|-----|----------|
| `ACTION` | string | Значение `UPDATE_GROUP`. Допустим алиас **`UPDATE_STATUS_GROUP`** — приводится к `UPDATE_GROUP` в `InboundActionDispatcher`. |
| `ID` | int | Идентификатор группы (`> 0`). |
| `ACTIVE` | string | Ровно `Y` или `N`. |

## Опциональные поля

| Поле | Тип | По умолчанию |
|------|-----|----------------|
| `NAME` | string | не меняет имя, если пусто после trim |
| `C_SORT` или `SORT` | int | `100`; если указано `≤ 0`, подставляется `100` |

## Успешный ответ (фрагмент)

- `success`: `1`
- `result`: строка с числовым ID группы (совместимость со старыми интеграциями)
- `data.ID`: числовой ID группы

## Ошибки

- Отсутствует или неверный `ID`
- `ACTIVE` не `Y`/`N`
- Текст ошибки обновления — в `error` из `$group->LAST_ERROR`
